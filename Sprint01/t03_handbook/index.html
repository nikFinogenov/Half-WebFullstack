DNS

- DNS records are used to translate domain names into IP addresses

- AAAA Record: Functions akin to A records but caters to IPv6 addresses.
- MX Record: Links a domain name to the mail exchange servers responsible for managing email communications for that domain.
- CNAME Record: Acts as an alias, redirecting one domain name to another, often employed for subdomains or simplifying URLs.
- TXT Record: Holds arbitrary text data, frequently utilized for verification or authentication purposes.
- SRV Record: Specifies the location of a service on a domain, such as a specific port for a particular protocol.
- NS Record: Specifies the authoritative name servers for a domain.

HTTP methods

HTTP methods are used to indicate the action

- GET: Requests a representation of a resource
- POST: Submits an entity to the specified resource
- PUT: Replaces the entity of the specified resource
- DELETE: Deletes the specified resource
- HEAD: Requests the header of a resource
- OPTIONS: Requests information about the communication options
- CONNECT: Converts the request connection to a transparent TCP/IP tunnel.
- TRACE: Echoes back the received request
- PATCH: Partially modifies the specified resource

HTTP status codes

HTTP status codes divided in five categories:

100 - 199: Informational responses
200 - 299: Successful responses
300 - 399: Redirection messages
400 - 499: Client error responses
500 - 599: Client error responses
